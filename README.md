# dudu-growth-system 嘟嘟个人成长系统
✨ Dudu Growth System - A cyberpunk-style personal growth tracking app with daily logging, attribute panel, reward shop, achievement system, custom tasks, point management, daily/weekly statistics comparison, and full bilingual (Chinese/English) support.

✨ 嘟嘟成长系统 - 一个赛博朋克风格的个人成长追踪应用。支持日志记录、属性面板、奖励商城、成就系统、自定义任务、清空积分、每日和每周积分对比，以及完整的中英文切换功能。
